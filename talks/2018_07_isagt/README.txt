Title:      Machine Learning for Compilers
Authors:    Chris Cummins
Presented:  Workshop on Introspective Systems for Automatically Generating
            Tests (ISAGT). Amsterdam, Netherlands, July 20th 2018.

"Software is eating the world, but AI is going to eat software". The meteoric
rise of machine learning promises enormous changes to how we write software. In
this talk I will provide a brief overview of some of ways machine learning has
been used to help both compiler developers and users, and outline some promising
new methodologies spearheaded by deep learning.
